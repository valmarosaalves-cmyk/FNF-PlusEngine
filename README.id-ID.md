## Perubahan baru yang keren pada Editor Chart dan perbaikan bug yang tak terhitung jumlahnya
![Chart Editor](docs/img/chart.png)
* Anda kini dapat membuat note "Event", yang berfungsi sebagai bookmark untuk memicu tindakan spesifik yang sebelumnya diprogram secara statis pada versi dasar permainan.
* BPM lagu Anda kini dapat memiliki nilai desimal
* Anda dapat secara manual menyesuaikan waktu strum sebuah note jika Anda menginginkan presisi milidetik
* Anda dapat mengubah jenis note di Editor, yang dilengkapi dengan lima contoh jenis:
  * Alt Animation: Memaksa animasi alternatif untuk diputar, berguna untuk lagu seperti Ugh/Stress
  * Hey: Memaksa animasi "Hey" alih-alih animasi Sing dasar. Jika Boyfriend memukul note ini, Girlfriend juga akan melakukan "Hey!".
  * Hurt Notes: Jika Boyfriend memukul note ini, dia akan memainkan animasi miss dan kehilangan beberapa kesehatan.
  * GF Sing: Alih-alih karakter memukul note dan bernyanyi, Girlfriend yang bernyanyi.
  * No Animation: Karakter hanya memukul note, tanpa animasi yang diputar.

 * Di Preferensi, Anda dapat mengaktifkan atau menonaktifkan Gulir Bawah, Gulir Tengah, Anti-Aliasing, Kecepatan Frame, Kualitas Rendah, Note Splashes, Lampu Berkedip, dan lain-lain.
![Options](docs/img/Options.png)

## Fitur gameplay lainnya:
* Saat musuh memukul note, note yang mereka petik juga akan berkilau.
* Lag tidak lagi memengaruhi pergerakan kamera dan skala ikon pemain.
* Beberapa perubahan berdasarkan pembaruan Minggu 7 telah ditambahkan (Warna latar belakang di Freeplay, Note splashes)
* Anda dapat mereset skor di Freeplay/Cerita Mode dengan menekan tombol Reset.
* Anda dapat mendengarkan lagu atau menyesuaikan Kecepatan Gulir/Kerusakan yang Diterima/dll. di Freeplay dengan menekan tombol Space.
* Anda dapat mengaktifkan "Penumpukan Kombo" di Opsi Gameplay. Ini menyebabkan sprite combo menjadi satu sprite dengan animasi dari pada sprite yang muncul setiap kali note dipukul.
